This template is for STM32F3DiscoveryKit. It has 8 on-board LEDs and 1 user buttons. With this code, LED4, LED5, LED8 and LED9 are on, and LED6 and LED7 are off, and LED10 is toggling. Push the user button to turn off LED3.

Connect BOOT0 with 3V and Press RESET button before you flash the code. 
Disconnect BOOT0 with 3V and Press RESET button again to run the code.
